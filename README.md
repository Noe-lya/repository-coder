# 🎁 Fabianidades - Souvenirs personalizados

Sitio web oficial de **Fabianidades**, un emprendimiento de souvenirs originales para eventos únicos. Ofrecemos productos como llaveros temáticos, straps, apagavelas y recuerdos para 15 años, bautismos, comuniones y 1° año.
## 📌 Descripción

Fabianidades es un proyecto creado por Ali y Meli (madre e hija), donde combinan creatividad y pasión por los detalles. La web permite conocer la marca, explorar productos, leer reseñas, contactar al equipo y simular un carrito de compras. Está desarrollada con **HTML5**, **CSS3**, **SASS**, **Bootstrap 5** y **AOS** (animaciones), con un diseño completamente responsive.

## 🚀 Características

- **Diseño responsive** que se adapta a móviles, tablets y escritorio.
- **Animaciones suaves** al hacer scroll con AOS.
- **Componentes de Bootstrap 5** (grid, botones, etc.).
- **Enlace directo a WhatsApp** flotante para consultas rápidas.
- **Video de fondo en bucle** en la sección principal.
- **Footer organizado** con enlaces a productos, redes sociales y contacto.
- **Estructura SASS** con archivos `.scss` y sourcemaps.
- **Páginas internas** totalmente funcionales (productos, reseñas, contacto, carrito).

## 🛠️ Tecnologías utilizadas

| Tecnología            | Uso                                         |
|-----------------------|---------------------------------------------|
| HTML5                 | Estructura del sitio                        |
| CSS3                  | Estilos base                                |
| SASS                  | Preprocesador CSS (archivo `style.scss`)    |
| Bootstrap 5           | Framework responsive y componentes          |
| AOS (Animate On Scroll) | Animaciones al desplazarse                |
| Google Fonts          | Fuentes `Faculty Glyphic` y `Ribeye`        |
| Node.js (implícito)   | Para gestionar dependencias (node_modules)  |

## 🔧 Instalación y uso local

Sigue estos pasos para ejecutar el proyecto en tu máquina:

1. **Clona o descarga** este repositorio.
2. Asegúrate de tener [Node.js](https://nodejs.org/) instalado (opcional, solo si quieres recompilar SASS).
3. Si deseas compilar SASS manualmente, ejecuta:
   ```bash
   sass --watch style.scss:style.css --style compressed
